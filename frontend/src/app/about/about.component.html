About This Project

A modern full-stack e-commerce application built with performance, scalability, and user experience in mind.

Project Overview

This project is a full-stack web application designed to demonstrate real-world development practices. It includes user authentication, product listings, cart management, and responsive UI components.

The goal was to build a clean, scalable, and maintainable application using modern technologies while following industry best practices.

🛠 Technologies Used

  • Angular – Frontend framework for building dynamic, component-based user interfaces
  • Django – High-level Python web framework for backend development
  • Django REST Framework – Toolkit for building secure and scalable REST APIs
  • Bootstrap – Responsive UI framework for fast and consistent design
  • CSS3 – Styling language used for layouts, animations, and responsiveness
  • Swiper.js – Modern touch slider for product carousels and animations
  • JWT Authentication – Secure token-based authentication for user sessions
